## 概述

VETO 是一款基于Svelte5的集现代军事推演、模拟联合国会议系统的综合平台，支持多军种作战单位的态势标绘、时间轴推演与 HOI4 风格战斗结算。无需安装，即开即用。

- **危机推演** — 自定义战场，部署陆海空三军单位，实时推演作战进程
- **会议模式** — 模拟联合国会议流程，支持多方代表发言、提案与表决
